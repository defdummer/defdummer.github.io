#The Windows Clipboard App

I use the windows clipboard. A lot. So I was really excited (and weeks later that excitement hasn't left me!) to discover the Windows Clipboard App!

The magic is this - Windows + V.

![Windows Clipboard App](https://github.com/defdummer/defdummer.github.io/blob/master/images/ClipboardAppSnapshot.PNG "Clipboard")

Now I have access to everything I have copied/cut since I started Windows. I love pasting something that I copied hours ago. 

If you have a Windows computer, I encourage you to try this app out.
